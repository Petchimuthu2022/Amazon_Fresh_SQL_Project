Designed and implemented a normalized relational database for Amazon Fresh, covering customers, products, suppliers, orders, order details, and reviews.
Wrote DDL statements to define tables with primary keys, foreign keys, UNIQUE, NOT NULL, CHECK, and DEFAULT constraints to enforce data integrity.
Performed core DML operations — INSERT, UPDATE, and DELETE — to manage product and supplier records.
Built aggregation and filtering queries using WHERE, GROUP BY, HAVING, and ORDER BY to analyze orders, ratings, and sales.
Used multi-table JOINs to calculate revenue per order and identify high-value customers.
Applied subqueries to find top-selling products and customers with no orders, and normalized the Products table to 3NF by extracting a Categories table.
Derived actionable business insights, including Prime member concentration by city and the top 3 most frequently ordered product categories.
